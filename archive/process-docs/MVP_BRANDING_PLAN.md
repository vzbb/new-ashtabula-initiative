# MVP Branding Plan — Targeted by Business / Organization / Lead

Goal: every MVP has a specific, named target (business, organization, or lead). No generic branding.

## Process
1) Identify the best-fit target for each MVP (real or intended buyer).
2) Update brand copy + visuals to reflect that target (hero, tagline, CTA, use-cases).
3) Log the target in a per‑MVP table (below) and in the project’s README/NOTES.
4) Ensure contact form or CTA references the target’s outcomes.
