# Branding Round Checklist

Use this checklist during periodic branding rounds.

## Per‑Site Audit
- [ ] Site has a clear **specific buyer/organization** name
- [ ] Branding matches that buyer (logo/style/colors/voice)
- [ ] Copy references the buyer’s real context/needs
- [ ] CTA routes to buyer‑specific outcome
- [ ] Footer/contact aligns with buyer

## If Unbranded / Generic
- [ ] Identify top 2–3 candidate buyers/leads
- [ ] Research each candidate (site, mission, services)
- [ ] Decide if multiple variants needed
- [ ] Duplicate site for each candidate
- [ ] Re‑skin aesthetics + copy to match each candidate
- [ ] Keep core functionality intact or enhance it

## Tracking
- [ ] Log findings + changes in `PROJECTS.md`
- [ ] Update any new lead research files

---

## Frequency
- Run weekly or after each new batch of SPAs.
