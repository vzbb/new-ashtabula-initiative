# Branding Asset Checklist — MVP Pages

Use this to ensure each MVP is clearly targeted to a specific business/org/lead.

- [ ] Target name is explicit (business/org/lead)
- [ ] Hero headline references target’s mission or sector
- [ ] Subhead states the exact pain solved
- [ ] CTA phrased for target (demo/audit/intake)
- [ ] 3 use‑case bullets tied to target’s workflow
- [ ] Testimonial or proof line (real or placeholder)
- [ ] Visual accent aligned to target (logo color / industry motif)
- [ ] Footer mentions target/region
