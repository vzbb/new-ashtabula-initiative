# MVP Targeting Questions (Business / Org / Lead)

Use these to lock in a specific target for each MVP before branding.

## Quick Questions
1) Who is the most likely **real buyer** for this MVP (business/org/lead)?
2) What is their **primary pain** this MVP solves?
3) What **proof point** should the homepage emphasize for that target?
4) What is the ideal **CTA** for that target (demo, audit, intake, call)?
5) Is this a **single‑target** or should we create **variants**?
