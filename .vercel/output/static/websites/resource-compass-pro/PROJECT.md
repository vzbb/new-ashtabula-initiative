# PROJECT — Benevolence Engine (Resource Compass Pro)

## Desired Functionality (from CSV)
- **Benevolence Engine / Resource Compass (G.O. Community Dev Corp):** Multilingual intake + resource finder that qualifies residents for aid programs and provides geo‑located directions to nearby food banks/shelters.

## Current Functionality (observed)
- User selects a need category and ZIP code.
- Gemini API returns 3 “local resources” with short descriptions.
- No map, routing, or verified resource database.

## Gaps
- No authoritative resource directory or eligibility screening logic.
- No multilingual UX, map, or directions.
- No intake workflow or referral tracking.

## High‑Priority Improvements
1. Build/ingest a vetted resource database with addresses, hours, eligibility rules.
2. Add map + directions (geo‑aware) and multi‑language support.
3. Add intake questionnaire and referral handoff tracking.
4. Add admin tools to update resources and monitor usage.

## Assumptions
- Treated as upgraded version of **Resource Compass / Benevolence Engine** CSV entries.
