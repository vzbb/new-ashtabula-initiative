# Resource Compass Pro

Professional resource matching tool for the Ashtabula Chamber of Commerce. Advanced intake and referral system for businesses, community organizations, and residents seeking local services.

## Features

- 🔍 **Smart Intake** - Quick ZIP + need-based search
- 🤖 **Gemini AI Matching** - Intelligent resource recommendations
- ⚡ **60-Second Process** - From search to referral in under a minute
- 🌐 **Geo-Aware Results** - Location-specific recommendations

## Resource Categories

- Food Assistance
- Shelter & Housing
- Utilities Support
- Mental Health Services
- Legal Aid

## Use Cases

- **Chamber Staff** - Member resource referrals
- **Business Owners** - Employee assistance programs
- **Nonprofits** - Client intake and routing
- **Community Navigators** - Professional resource matching

## Tech Stack

- React 18 + Vite
- Google Gemini 1.5 Flash API
- Custom Pro Compass SVG logo
- Lato + Merriweather typography

## Environment Variables

```bash
VITE_GEMINI_API_KEY=your_gemini_api_key
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Demo Script

1. Enter client ZIP code (e.g., 44004)
2. Select need type (e.g., "Utilities")
3. Click "Find Resources"
4. AI generates referral list with descriptions
5. Option to add to client intake record
