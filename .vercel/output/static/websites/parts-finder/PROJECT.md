# Parts Finder

**Status:** 🟡 Phase 1 Complete (Research)  
**Priority:** P2 (Automotive/Business Tools)  
**Target:** DIY mechanics and repair shops in Ashtabula County

---

## Problem
Finding auto parts requires calling 3-5 stores. Wasted trips when parts not in stock. No price comparison between stores. Special order delays with no visibility.

## Solution
"Kayak for auto parts" — search all local stores at once, compare prices and availability, reserve for pickup or request delivery.

## Key Differentiators
- Universal search across all local parts stores
- Real-time inventory availability
- Price comparison for local pickup
- VIN-based fitment verification
- Support local businesses (vs. RockAuto/Amazon)

## Market
- 65,000-75,000 registered vehicles in Ashtabula County
- 8,000-12,000 DIY mechanics
- 30-50 repair shops
- $15M-$25M annual parts spend
- 8-12 parts stores (AutoZone, Advance, O'Reilly, NAPA, etc.)

## Revenue Model
| Stream | Rate |
|--------|------|
| Affiliate commission | 5-10% (online orders) |
| Lead fee | $1-3 (in-store pickup) |
| Delivery fee | $5-10 |
| Featured listing | $50-100/mo |
| Shop subscription | $29-79/mo |

## Tech Stack (Proposed)
- Next.js + Tailwind CSS
- Supabase (database)
- CarQuery/NHTSA API (fitment data)
- Google Maps API (store locations)
- Stripe (payments)
