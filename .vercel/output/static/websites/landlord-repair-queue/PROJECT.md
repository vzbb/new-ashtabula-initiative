# Landlord Repair Queue

**Status:** 🟡 Phase 1 Complete (Research)  
**Priority:** P2 (Business Tools)  
**Target:** Property managers and landlords in Ashtabula County

---

## Problem
Property managers spend 10-15 hours/week on maintenance coordination. Tenants frustrated with no status updates. Phone tag between tenants, landlords, and contractors. No system for tracking urgent vs. routine repairs.

## Solution
"Zendesk for property maintenance" — tenant request submission, manager triage and dispatch, contractor completion tracking, everyone stays informed via mobile app.

## Key Differentiators
- Unified maintenance queue (not scattered texts/emails)
- Photo documentation (before/after)
- Tenant-facing mobile app with status tracking
- Local Ashtabula contractor network
- Affordable for small landlords ($29/mo vs. $200+ for AppFolio)

## Market
- 12,000-15,000 rental units in Ashtabula County
- 30,000-60,000 annual maintenance requests
- 100-150 independent landlords (5-20 units)
- 15-25 professional property management companies
