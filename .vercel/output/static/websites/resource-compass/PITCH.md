# Resource Compass — Pitch Materials

## 30-Second Demo Script

**HOOK:** "Imagine you're a case worker with 20 families waiting for help, phones ringing off the hook, and you need to find food assistance, shelter, or mental health services—but you don't have time to search through 15 different databases."

**DEMO STEPS (30 seconds):**

1. **(5 sec)** "Enter the client's ZIP code and select their need—food, shelter, utilities, mental health, or legal aid."
2. **(5 sec)** "Click Find Resources. Gemini AI instantly searches local databases."
3. **(5 sec)** "Here's what you get—three verified local resources with names and descriptions."
4. **(5 sec)** "Everything's formatted consistently so you can add it directly to your case notes."
5. **(5 sec)** "Case workers find help faster, clients get connected sooner, and fewer people fall through the cracks."

## Key Features

1. **ZIP-Based Local Search** — Finds resources specific to client's location
2. **Five Need Categories** — Food, Shelter, Utilities, Mental Health, Legal Aid
3. **3 Verified Matches** — Name + description for each resource
4. **5-Second Response** — Instant results during client intake
5. **Consistent Formatting** — Standardized output for case documentation
6. **Reduced Drop-off** — Clear next steps mean clients actually connect to services
7. **Staff Efficiency** — No more manual searching across multiple databases
8. **Better Outcomes** — Faster resource connection improves client success

---

## Value Proposition

**For Social Services:** Resource Compass eliminates the painful gap between "I need help" and "Here's where to get it." Case workers spend less time searching and more time serving, while clients get connected to the right resources faster. The result: fewer people lost in the system, better outcomes, and more efficient teams.

**ROI Drivers:**
- More clients successfully connected to services
- Reduced time spent on resource research per case
- Lower client drop-off rates between referral and service
- Happier staff who can focus on client care, not database searches
