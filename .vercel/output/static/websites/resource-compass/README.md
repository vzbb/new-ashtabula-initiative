# Resource Compass

AI-powered resource finder for Ashtabula County Rural Support Services. Connects residents with local assistance programs, social services, and community support.

## Features

- 🧭 **Smart Resource Matching** - Find help by ZIP code and need type
- 🤖 **Gemini AI Search** - AI-curated local resource recommendations
- 🏥 **Comprehensive Coverage** - Food, shelter, utilities, health, legal, jobs, childcare
- 📍 **Location-Aware** - Results tailored to specific Ashtabula County areas

## Available Resource Categories

- Food Assistance
- Shelter & Housing
- Utilities Help
- Mental Health Services
- Legal Aid
- Job Training
- Healthcare
- Childcare

## Use Cases

- **Residents in Crisis** - Quickly find emergency assistance
- **Social Workers** - Resource referrals for clients
- **Community Navigators** - Guide residents to appropriate services
- **First Responders** - Connect people with follow-up support

## Tech Stack

- React 18 + Vite
- Google Gemini 1.5 Flash API
- Custom County Seal SVG logo
- CSS3 with county branding colors

## Environment Variables

```bash
VITE_GEMINI_API_KEY=your_gemini_api_key
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Demo Script

1. Enter ZIP code (e.g., 44004 for Ashtabula)
2. Select need type (e.g., "Food Assistance")
3. Click "Find Resources"
4. AI returns 3 local resources with contact info
