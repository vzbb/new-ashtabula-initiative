# PROJECT — Destination Navigator (MyTrip Planner Export)

## Desired Functionality (from CSV)
- **MyTrip Planner / Destination Navigator (Ashtabula CVB):** Chatbot that builds custom 3‑day itineraries (e.g., kid‑friendly + wineries), helping visitors plan logistics and spend more locally.

## Current Functionality (observed)
- Inputs for theme and number of days.
- Gemini API returns a short day‑by‑day itinerary text.
- No real attraction database, maps, or export functionality beyond plain text.

## Gaps
- No curated business/attraction database or booking links.
- No itinerary export (PDF/email) or shareable plan.
- No personalization beyond theme/days; no budget/party size constraints.

## High‑Priority Improvements
1. Connect to local attractions/events database with hours, booking, and partner promotions.
2. Add map view and export/share (PDF, email, print).
3. Expand inputs (party type, budget, accessibility, dates) for better personalization.
4. Track clicks/partner referrals for ROI reporting.

## Assumptions
- Mapped to **Ashtabula CVB** CSV entry.
