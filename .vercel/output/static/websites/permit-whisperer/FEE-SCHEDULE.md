## City of Ashtabula — Zoning Permit Fees

### Residential Construction

| Permit Type | Fee Structure | Est. Cost Examples |
|------------|---------------|-------------------|
| Zoning Permit (Residential) | 1% of total project value | $100K project = $1,000 |
| Zoning Permit (Commercial) | 1% of total project value + plan review | Contact PCD for commercial |
| Shed exemption | $0 | <200 sq ft, height limits apply |
| Deck exemption | $0 | <30" above grade |
| Fence permit | $50 flat fee | Per property line segment |
| Pool permit | $150 base + $50/safety inspection | Typical total: $200-250 |
| Demolition permit | $100 base + $25/structure | House demo: ~$125 |
| Sign permit | $75 base + $25/sign face | Varies by sign type |
| Temporary use permit | $100 flat fee | 30-day max, renewable |
| Home occupation permit | $75 annual | Renewed yearly |

### Fee Calculations

**Project Value Definition:**
- New construction: Total construction cost (materials + labor)
- Additions: Cost of addition only
- Renovations: Cost of work being performed
- Owner-builder: Must provide material receipts + labor estimates

**Minimum Fees:**
- Residential zoning permit: $50 minimum
- Commercial zoning permit: $150 minimum

**Maximum Fees:**
- Residential: $2,500 cap
- Commercial: $5,000 cap

### Additional City Fees

| Fee Type | Amount | When Applied |
|----------|--------|--------------|
| Plan review fee | 25% of permit fee | Commercial projects only |
| Reinspection fee | $50 per visit | Failed inspections |
| Permit extension | $50 | 6-month extension request |
| Permit cancellation | $40 | If work not started |
| Permit reactivation | $75 | Lapsed permit restart |
| After-hours inspection | $100 | Weekend/holiday requests |
| Duplicate permit card | $15 | Lost permit replacement |

### Historic District Surcharge

Properties in the Historic District require additional review:

| Review Type | Fee | Timeline |
|------------|-----|----------|
| Historic Review Certificate | $100 | 14-day review |
| Certificate of Appropriateness | $150 | 30-day review |
| Minor work waiver | $25 | 7-day review |

**Combined Example:** Historic District home addition ($150K project)
- Zoning permit (1%): $1,500
- Historic Review Certificate: $100
- **Total City fees: $1,600**

## Fee Exemptions & Reductions

### Automatic Exemptions (No Fee)
- Sheds ≤200 sq ft (City only)
- Decks ≤30" above grade (City only)
- Fences under 6 feet (agricultural areas only)
- Government-owned properties
- Public utility emergency repairs

### Potential Reductions (Case-by-Case)
- Non-profit organizations (50% reduction possible)
- Senior/disabled homeowner programs (City only; inquire with PCD)
- Emergency repairs (waiver of rush fees)
- Multiple permits same property (bundle discount possible)

## Common Fee Questions

**Q: Why do I pay twice for the same project?**
A: City zoning permits ensure compliance with land use regulations (setbacks, height, use). County building permits ensure compliance with Ohio Building Code (safety, structural). They serve different purposes and are legally separate.

**Q: Can I get a refund if I cancel?**
A: City: $40 cancellation fee applies if work hasn't started. County: Contact Building Department for refund policy.

**Q: What if my project value changes?**
A: If actual costs exceed estimated by >20%, you must amend the permit and pay the difference. If under, no refund typically issued.

**Q: Are fees the same for DIY vs. contractor work?**
A: Yes, fees are based on project value regardless of who does the work. However, DIY projects must still provide realistic cost estimates.
