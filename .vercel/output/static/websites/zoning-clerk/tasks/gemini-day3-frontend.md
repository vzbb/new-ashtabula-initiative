# Zoning Clerk MVP - Day 3: React Frontend

## Task for Gemini CLI
Build a React + TypeScript + Vite SPA frontend:

## Components to create:
1. **PropertyLookup** - Address input with autocomplete, map integration placeholder
2. **ZoningResult** - Display zoning district, allowed uses, restrictions
3. **ChatInterface** - RAG-powered chat with citation support
4. **PermitWizard** - Step-by-step permit requirement flow
5. **DocumentViewer** - Display relevant code sections

## Pages:
- / - Home with property lookup
- /zoning/:address - Results page
- /chat - Chat interface
- /permits - Permit wizard

## Design system (Ashtabula branding):
- Primary: #1e3a5f (deep Lake Erie blue)
- Secondary: #c9a227 (bridge gold/amber)
- Accent: #2d5a87 (lighter blue)
- Background: #f8f9fa (clean white/gray)
- Typography: Inter or system fonts
- Rounded corners: 8px
- Shadows: subtle, professional

## Features:
- Responsive (mobile-first)
- Loading states
- Error boundaries
- Dark mode toggle

## Tech:
- React 18, TypeScript, Vite
- React Router
- Tailwind CSS (or styled-components)
- Lucide React icons
