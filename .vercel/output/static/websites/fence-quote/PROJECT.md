# Fence Quote

**Status:** 🟡 Phase 1 Complete (Research)  
**Priority:** P2 (Business Tools)  
**Target:** Fencing contractors in Ashtabula County

---

## Problem
Fencing contractors waste time on unqualified site visits. Homeowners want instant pricing transparency. Current process involves phone tag and delayed quotes.

## Solution
Instant online quoting tool with interactive property mapping, material comparison, and qualified lead generation for fencing contractors.

## Key Differentiators
- Visual fence line drawing on satellite imagery
- Local Ashtabula pricing (not generic estimates)
- Qualified leads (homeowners see price before contacting)
- Material comparison tool with visual preview

## Market
- 800-1,200 residential fence installs/year in county
- $2.4M - $9.6M annual addressable market
- 10-15 local fencing contractors identified
- Pool installations (code-required fencing) drive demand

## Revenue Model
| Tier | Price | Target |
|------|-------|--------|
| Free | $0 | New contractors (5 quotes/mo) |
| Pro | $79/mo | Solo operators |
| Business | $199/mo | 5+ employee companies |
| Enterprise | Custom | Regional chains |

## Tech Stack (Proposed)
- Next.js + Tailwind CSS
- Supabase (database + auth)
- Google Maps API or Mapbox
- Vercel hosting
