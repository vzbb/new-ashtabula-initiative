# PROJECT — Parts Finder Request

## Desired Functionality (from CSV)
- **Parts Finder AI (Ace Parts Company):** Chat interface to lookup inventory for specific vehicle parts and confirm availability.

## Current Functionality (observed)
- Input for free‑text query.
- Gemini API generates a confirmation response and asks to confirm fitment.
- No inventory lookup or pricing.

## Gaps
- No real inventory/fitment database or SKU lookup.
- No pricing, availability, or reservation workflow.
- No order capture or lead tracking.

## High‑Priority Improvements
1. Connect to inventory system with SKU, fitment, and availability data.
2. Add structured input for vehicle year/make/model and part category.
3. Provide pricing + reserve/hold workflow and staff notifications.
4. Capture lead info and integrate with sales follow‑up.

## Assumptions
- Directly mapped to **Ace Parts Company** CSV entry.
