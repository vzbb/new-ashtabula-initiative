# PROJECT — SommelierBot Pro (Pocket Sommelier)

## Desired Functionality (from CSV)
- **Pocket Sommelier / SommelierBot Pro (Ferrante Winery & Ristorante):** QR/tablet menu that recommends wine pairings for selected dishes; boosts upsell.

## Current Functionality (observed)
- Input for dish name.
- Gemini API returns pairing + 1‑sentence upsell.
- No integration with real menu, wine list, or POS.

## Gaps
- No structured menu or wine inventory; recommendations are generic.
- No QR/tablet flow, table context, or staff mode.
- No analytics on upsells or conversion.

## High‑Priority Improvements
1. Load actual menu and wine list (with availability, price tiers, margins).
2. Add QR/tablet UX with table selection and branded style.
3. Add analytics for upsell performance and staff prompts.
4. Provide feedback loop for wine managers to tune recommendations.

## Assumptions
- Mapped to **Ferrante Winery** SommelierBot Pro CSV entry.
