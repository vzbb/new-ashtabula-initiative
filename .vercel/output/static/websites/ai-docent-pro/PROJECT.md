# PROJECT — AI Docent Pro

## Desired Functionality (from CSV)
- **AI Docent (Ashtabula Maritime Museum):** Visitors snap photos of artifacts; AI narrates history (with TTS like F5TTS).

## Current Functionality (observed)
- Textarea for artifact description.
- Gemini API generates 120‑word narration plus 5 tags.
- No image input, no audio output, no artifact database.

## Gaps
- Missing photo capture/upload and image understanding pipeline.
- No TTS or audio playback; no kiosk/QR experience.
- No curated artifact metadata or approval workflow.

## High‑Priority Improvements
1. Add photo upload or camera capture + vision model for artifact identification.
2. Integrate TTS playback and downloadable audio guides.
3. Connect to curated artifact database with verified facts and citations.
4. Provide admin moderation and on‑site display mode.

## Assumptions
- Directly mapped to **AI Docent** CSV entry.
