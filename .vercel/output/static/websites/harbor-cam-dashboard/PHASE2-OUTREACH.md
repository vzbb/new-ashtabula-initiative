## 3. Data Integration Plan

### NOAA Buoy 45005 (Primary Data Source)

**Data Available:**
- Wind speed and direction
- Wave height and period
- Water temperature
- Air temperature
- Barometric pressure

**API Endpoint:**
```
https://www.ndbc.noaa.gov/station_realtime.php?station=45005
```

**Update Frequency:** Hourly (or more frequent during events)

**Implementation:**
- Scrape or API (NOAA provides RSS/JSON)
- Cache for performance
- Display in dashboard widget

### Camera Integration

**Redbrook LakeVision:**
- Current URL: redbrookboatclub.com/lakevision-camera.html
- May require iframe embed or direct stream URL
- Contact for permission and technical specs

**Future Cameras:**
- Harbor Yacht Club (permission needed)
- Ashtabula Yacht Club (permission needed)
- City of Ashtabula (partnership needed)
- Private marina cams (outreach)

### Weather Overlay

**OpenWeatherMap API:**
- Current conditions
- Hourly forecast
- Weather alerts
- Free tier: 1,000 calls/day

**NOAA Weather API:**
- Official forecasts
- Marine forecasts
- Free

## 4. Dashboard Design Specifications

### Mobile-First Layout

```
┌─────────────────────────────────┐
│  ASHTABULA HARBOR CONDITIONS    │
├─────────────────────────────────┤
│  [LIVE CAMERA FEED]             │
│  Redbrook Boat Club Camera      │
│  Last updated: 10:32 AM         │
├─────────────────────────────────┤
│  CURRENT CONDITIONS             │
│  ┌──────────┬──────────┐       │
│  │ Wave Ht  │ 1.2 ft   │       │
│  │ Water    │ 72°F     │       │
│  │ Wind     │ 8 mph NW │       │
│  └──────────┴──────────┘       │
├─────────────────────────────────┤
│  STATUS: ✅ GOOD TO GO          │
│  Conditions favorable for       │
│  recreational boating.          │
├─────────────────────────────────┤
│  TREND: Improving ↗             │
│  Next 3 hours: Remaining calm   │
├─────────────────────────────────┤
│  [Local Charters] [Weather]     │
└─────────────────────────────────┘
```

### Safety Indicators

| Wave Height | Status | Recommendation |
|-------------|--------|----------------|
| 0-2 ft | ✅ GOOD | Safe for all boaters |
| 2-4 ft | ⚠️ CAUTION | Small craft advisory |
| 4+ ft | ❌ NO-GO | Dangerous conditions |

### Color Coding

- **Green:** Wave height 0-2 ft, wind <15 mph
- **Yellow:** Wave height 2-4 ft, wind 15-25 mph
- **Red:** Wave height >4 ft, wind >25 mph

## 5. Revenue Model Options

### Option 1: Free Community Service (Recommended for MVP)
- All features free
- Funded by tourism grants, marina sponsorships
- Community goodwill approach
- Donation button for support

### Option 2: Freemium
- **Free:** Current conditions, basic camera
- **Premium ($2.99/mo or $19.99/yr):**
  - Historical data/trends
  - SMS alerts for conditions
  - Multiple camera views
  - Fishing reports integration
  - Ad-free experience

### Option 3: Sponsorship
- Marinas/businesses sponsor camera maintenance
- Logo placement on dashboard
- "Brought to you by [Sponsor]"
- Annual sponsorship tiers

### Option 4: Data Licensing
- Aggregate condition data
- Sell to fishing apps, tourism researchers
- Anonymized, trend-level data only

## 6. Risk Assessment Matrix

| Risk | Likelihood | Impact | Mitigation |
|------|------------|--------|------------|
| Camera owner won't partner | Medium | HIGH | Have backup camera sources, offer value |
| Camera goes offline | Medium | MEDIUM | Fallback message, multiple sources |
| NOAA buoy data unavailable | Low | MEDIUM | Cached data, weather backup |
| Low user adoption | Medium | MEDIUM | Partner with marinas, fishing groups |
| Liability for safety decisions | Medium | HIGH | Clear disclaimers, "reference only" |
