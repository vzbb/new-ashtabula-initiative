### Google Maps API Cost Analysis

**Current Pricing (as of Feb 2026):**
- Static Maps: $2 per 1,000 requests
- Geocoding: $5 per 1,000 requests
- Places API: $17 per 1,000 requests

**Estimated Cost per Quote:**
- Property lookup (geocoding): $0.005
- Static map display: $0.002
- **Total: ~$0.007 per quote**

**Monthly Estimate (50 quotes):** $0.35  
**Monthly Estimate (500 quotes):** $3.50  
**Conclusion:** API costs are negligible; viable for free tier

### Alternative: Mapbox
- Free tier: 50,000 loads/month
- Beyond free: $0.50 per 1,000
- **More generous free tier than Google**

**Recommendation:** Start with Mapbox for free tier sustainability

## 5. Competitive Intelligence — Calculator UX Audit

### LawnGuru Quote Flow
1. Enter ZIP code
2. Select service type (mow, fertilize, etc.)
3. Enter lot size OR use satellite measurement
4. See instant price
5. Book immediately

**Strengths:** Fast, clean, satellite measurement  
**Weaknesses:** Forces booking through platform; no contractor branding

### TaskEasy Quote Flow
1. Enter address
2. System measures via satellite
3. Select services
4. See price
5. Schedule service

**Strengths:** Fully automated measurement  
**Weaknesses:** No contractor relationship; high commission

### Jobber Calculator (Lead Gen)
1. Enter ZIP code
2. Select property type
3. See price RANGE (not exact)
4. "Get exact quote" → Jobber software pitch

**Strengths:** Simple, fast  
**Weaknesses:** Vague pricing; lead gen only

### Our Differentiation
- Exact pricing (not ranges)
- Contractor branding throughout
- Optional satellite OR manual entry
- No platform lock-in
- Free or low-cost

## 6. Go/No-Go Decision Tree

```
After Phase 2 outreach:
│
├─► If 5+ contractors express interest:
│   └─► Proceed to Phase 3 (SPEC.md)
│       └─► Prioritize their requested features
│
├─► If 2-4 interested but pricing concerns:
│   └─► Adjust pricing model
│       └─► Re-test with revised model
│
├─► If interested but need white-glove setup:
│   └─► Add setup service to plan
│       └─► Price accordingly
│
└─► If <2 interested or active opposition:
    └─► Pause
        └─► Document learnings, pivot or deprioritize
```

### Go Criteria
- [ ] 5+ contractors interested in pilot
- [ ] 60%+ prefer same pricing model
- [ ] <30% require extensive hand-holding
- [ ] No legal/regulatory barriers identified

### No-Go Criteria
- [ ] <2 contractors interested
- [ ] Active opposition from established players
- [ ] Overwhelming preference for existing solutions
- [ ] Technical barriers insurmountable
