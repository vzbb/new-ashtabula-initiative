## 4. Competitive Audit Checklist — FishingBooker

**Objective:** Understand captain experience on incumbent platform

### Sign-up & Onboarding Audit
- [ ] Create captain account (document friction points)
- [ ] Profile setup process (time required, required fields)
- [ ] Verification process (USCG license, insurance)
- [ ] Calendar integration options

### Pricing & Fee Structure
- [ ] Commission percentage tiers
- [ ] Payment processing fees
- [ ] When fees are deducted (booking vs. trip completion)
- [ ] Cancellation refund policy

### Captain Dashboard Analysis
- [ ] Booking management interface
- [ ] Customer communication tools
- [ ] Review/rating system mechanics
- [ ] Earnings reporting
- [ ] Mobile experience

### Weaknesses to Exploit
| Weakness | Our Advantage |
|----------|---------------|
| High commission (10-20%) | 0% with subscription model |
| National focus (not local) | Ashtabula-specific features |
| Generic platform | Community fishing reports, local conditions |
| No lodging integration | Package deals with local B&Bs |
| Impersonal | Local support, knows captains by name |

## 5. Legal & Compliance Research

### Questions for Attorney Consultation (1-hour session)

**Platform Liability**
- [ ] Does a booking platform need liability insurance?
- [ ] What disclaimers/terms of service are required?
- [ ] Captain independent contractor vs. employee classification risks

**Payment Processing**
- [ ] Ohio sales tax applicability on charter bookings
- [ ] Stripe Connect marketplace compliance requirements
- [ ] Chargeback liability allocation

**Regulatory Compliance**
- [ ] USCG licensing verification requirements
- [ ] Fishing license disclosure obligations
- [ ] Data privacy (customer info handling)

**Recommended Attorney Search**
- Maritime/boating law specialist OR
- Tech startup attorney with marketplace experience
- Budget: $300-500 for initial consultation
