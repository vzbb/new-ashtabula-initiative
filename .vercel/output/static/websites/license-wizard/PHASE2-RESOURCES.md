## Executive Summary

This document catalogs all license/permits required for businesses in Ashtabula, Ohio across federal, state, county, and city levels. Includes direct form links, fee schedules, contact information, and prerequisite chains.

## 1. Federal Level Requirements

### All Businesses (if applicable)

| Requirement | Business Types | Source | Cost | Form/Link |
|-------------|----------------|--------|------|-----------|
| **EIN (Tax ID)** | All except sole proprietors | IRS | Free | [IRS.gov EIN Assistant](https://www.irs.gov/businesses/small-businesses-self-employed/apply-for-an-employer-identification-number-ein-online) |
| **Alcohol (TTB)** | Breweries, distilleries, bars | TTB | Varies | [TTB.gov](https://www.ttb.gov/) |
| **Agriculture** | Farms, food processing | USDA | Varies | [USDA.gov](https://www.usda.gov/) |
| **Firearms (ATF)** | Gun shops | ATF | Varies | [ATF.gov](https://www.atf.gov/) |
| **DOT/FMCSA** | Trucking, logistics | DOT | Varies | [FMCSA](https://www.fmcsa.dot.gov/) |

### Key Federal Notes
- **EIN is FREE** — never pay for EIN services (avoid third-party filing sites)
- Alcohol permits take 60-90 days — plan accordingly
- Federal requirements are industry-specific, not universal

## 3. Ashtabula County Level Requirements

### 3.1 County Auditor — Vendor Licensing

| License Type | Cost | Renewal | Contact | Location |
|--------------|------|---------|---------|----------|
| **Regular Vendor License** | $50 | None (one-time) | 440-576-3786 | 25 West Jefferson St, Jefferson, OH 44047 |

**Application:** [Download Regular Vendor License Application (PDF)](/DNN/Portals/0/Regular%20Vendor%20License%20Application%202025.pdf)

**Notes:**
- Effective April 9, 2025: fee increased to $50 (was $25)
- **No annual renewal fee**
- Required for each fixed business location
- If selling by weight/volume: must contact Weights & Measures at 440-576-3791
- Hours: 8:00am - noon, 1pm - 4:30pm
- Payment: Cash, check, or money order

### 3.2 Ashtabula County Building Department

| Permit Type | Contact | Phone | Notes |
|-------------|---------|-------|-------|
| **Commercial Building Permits** | Building Dept | 440-576-9090 | Required for construction/renovation |
| **Residential Building Permits** | Building Dept | 440-576-9090 | New construction, additions |
| **Zoning Verification** | Varies by township | - | Contact local zoning authority |

**Address:** 25 W. Jefferson St., Jefferson, OH 44047  
**Hours:** Monday - Friday | 8 a.m. - 4:30 p.m.

**Forms:**
- Commercial Building Application: [Download PDF](https://www.ashtabulacounty.us/DocumentCenter/View/109/Commercial-Building-Application)

### 3.3 Ashtabula County Health Department — Food Service

| License Type | Risk Level | Inspections/Year | Contact |
|--------------|------------|------------------|---------|
| **Food Service Operation** | I-IV | 1-4 inspections | 440-576-6010 (option 3) |
| **Retail Food Establishment** | Varies | 1-2 inspections | 440-576-6010 (option 3) |
| **Mobile Food Service** | Varies | 1 inspection | 440-576-6010 (option 3) |
| **Temporary Food Service** | Event-based | 1 per event | 440-576-6010 (option 3) |

**Address:** 12 West Jefferson St., Jefferson, OH 44047

**Risk Levels Defined:**
- **Risk I:** Pre-packaged foods, fountain drinks (lowest risk)
- **Risk II:** Heat treating, holding food at temperature
- **Risk III:** Cutting raw meat, cooking/cooling food
- **Risk IV:** Reheating bulk quantities, catering, high-risk clientele

**Fees:** Check or money order only (made payable to Ashtabula County Health Department)
- Applications not refundable
- Fee charged for returned checks

**Regulations:**
- [Ohio Revised Code Chapter 3717](https://codes.ohio.gov/ohio-revised-code/chapter-3717)
- [Ohio Administrative Code Chapter 3717-1](https://codes.ohio.gov/ohio-administrative-code/chapter-3717-1)

## 8. Wizard Data Model

### Business Profile Schema

```javascript
{
  businessType: String,        // restaurant, retail, contractor, home-based, etc.
  locationType: String,        // home-based, commercial, mobile
  hasEmployees: Boolean,
  sellsProducts: Boolean,
  servesFood: Boolean,
  sellsAlcohol: Boolean,
  constructionWork: Boolean,
  professionalService: Boolean,
  entityType: String           // LLC, Corporation, Sole Proprietor, Partnership
}
```

### License Requirement Schema

```javascript
{
  id: String,
  name: String,
  level: String,               // federal, state, county, city
  authority: String,
  authorityContact: Object,
  cost: {
    amount: Number,
    type: String               // fixed, variable, free
  },
  formUrl: String,
  isRenewable: Boolean,
  renewalPeriod: String,       // annual, one-time, etc.
  prerequisites: [String],     // IDs of required prior licenses
  businessTypes: [String],     // Applicable business types
  description: String
}
```
