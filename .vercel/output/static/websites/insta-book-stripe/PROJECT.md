# PROJECT — Insta‑Book (Stripe)

## Desired Functionality (from CSV)
- **Insta‑Book Page (Christopher’s Dockside):** Mobile booking calendar linked to Stripe checkout for cottage rentals.

## Current Functionality (observed)
- Inputs for check‑in date, number of nights, and notes.
- Gemini API writes a confirmation message with an upsell line.
- No calendar availability or payment processing.

## Gaps
- No live availability calendar or inventory management.
- No Stripe checkout or deposit flow.
- No booking confirmation email/SMS or admin controls.

## High‑Priority Improvements
1. Add real availability calendar and booking hold/confirmation logic.
2. Integrate Stripe Checkout for deposits/fees and automated receipts.
3. Send confirmation email/SMS and allow admin booking management.
4. Collect guest contact info and policies (cancellation, check‑in rules).

## Assumptions
- Directly mapped to **Christopher’s Dockside** CSV entry.
