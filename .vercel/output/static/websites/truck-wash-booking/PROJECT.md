# Truck Wash Booking

**Status:** 🟡 Phase 1 Complete (Research)  
**Priority:** P2 (Business Tools)  
**Target:** Fleet operators and truck washes in Ashtabula County

---

## Problem
Truck drivers waste 1-2 hours waiting at wash facilities. No booking system exists. Fleet managers can't coordinate washes for multiple trucks. No price transparency or quality ratings.

## Solution
"OpenTable for truck washing" — book ahead, skip the line, know your price. Fleet dashboard for coordinating multiple trucks.

## Key Differentiators
- Advance booking (skip the wait)
- Price transparency upfront
- Fleet coordination dashboard
- Quality ratings from truckers
- Route planning with wash stops

## Market
- 2,500-3,500 commercial trucks in county
- 75,000-150,000 annual washes
- $3M - $7.5M annual market value
- Blue Beacon (only 24/7 option) always busy

## Revenue Model
| Stream | Rate |
|--------|------|
| Booking fee | $2-5 per wash |
| Commission | 5-10% of wash cost |
| Fleet subscription | $49/mo |
| Priority placement | $25/mo |

## Tech Stack (Proposed)
- Next.js + Tailwind CSS
- Supabase (database)
- Stripe Connect (payments)
- Google Maps API
- Twilio (SMS notifications)
