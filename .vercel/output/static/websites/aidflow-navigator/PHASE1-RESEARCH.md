# AidFlow Navigator — Phase 1 Research & Recon
**Date:** February 17, 2026  
**Status:** 🟡 Research Initiated  

---

## Problem Statement

Low-income families in Ashtabula County struggle to:
1. Discover available assistance programs
2. Understand eligibility requirements
3. Navigate application processes
4. Find contact information for aid organizations

**Result:** Many eligible families don't access benefits they're entitled to.

## Competitor Analysis

### Existing Tools

| Tool | Approach | Gap |
|------|----------|-----|
| **Benefits.gov** | Federal program finder | Not locally focused |
| **211 Helpline** | Phone-based referral | No self-service web tool |
| **County websites** | PDF lists, scattered info | Hard to navigate, not personalized |
| **Non-profit sites** | Individual org info | No unified view |

### Key Finding
**No unified, user-friendly tool** exists specifically for Ashtabula County assistance programs with eligibility screening.

## User Personas

### Persona 1: Single Parent
**Goal:** Get help with rent and utilities after job loss  
**Pain Points:** Don't know what's available, embarrassed to ask, complex applications  
**Needs:** Anonymous screening, clear eligibility, step-by-step guidance

### Persona 2: Senior on Fixed Income
**Goal:** Reduce prescription and heating costs  
**Pain Points:** Confusing paperwork, transportation barriers, phone-only options  
**Needs:** Simple interface, printable guides, contact info in one place

### Persona 3: Case Worker
**Goal:** Help clients find all available resources  
**Pain Points:** Information scattered, eligibility rules complex, programs change  
**Needs:** Comprehensive database, eligibility matrix, referral tool
