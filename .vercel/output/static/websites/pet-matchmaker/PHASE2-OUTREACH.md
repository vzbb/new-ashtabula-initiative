## 4. User Research Plan

### Survey: Adopter Preferences & Matching (Target: 25 responses)

**Distribution Channels:**
- Ashtabula Community News Facebook Group (12K members)
- Ashtabula County Chat Facebook Group (8K members)
- ACAPL Facebook page (partner request)
- Local library bulletin boards (physical QR code)

**Survey Questions:**

1. **Screening:** Are you currently looking to adopt a pet, considering it in the next 6 months, or just browsing? (Required)
   - Actively looking
   - Considering within 6 months
   - Just browsing/curious
   - Not interested

2. **Discovery:** How do you currently search for adoptable pets? (Select all)
   - Petfinder
   - Adopt-a-Pet
   - Facebook
   - ACAPL website
   - Other shelter websites
   - Word of mouth
   - Haven't searched

3. **Pain Point:** What's the biggest challenge when searching for a pet to adopt?
   - Too many options, hard to choose
   - Can't tell personality from photos
   - Don't know which shelters have what
   - Scheduling meet-and-greets
   - Other: ______

4. **Matching Interest:** Would you complete a 10-15 question quiz about your lifestyle if it helped match you with compatible pets?
   - Definitely yes
   - Probably yes
   - Not sure
   - Probably not
   - Definitely not

5. **Key Factors:** Rate importance for pet matching (1-5 scale):
   - Energy level (couch potato vs high energy)
   - Good with children
   - Good with other pets
   - Training level
   - Size
   - Age
   - Breed

6. **Return Concern:** How worried are you about adopting a pet that doesn't fit your lifestyle and needing to return it?
   - Very worried
   - Somewhat worried
   - Neutral
   - Not very worried
   - Not worried at all

7. **Features:** Which features would you use? (Select all)
   - Personality matching quiz
   - See all Ashtabula pets in one place
   - Save favorite pets
   - Get alerts for new matches
   - Schedule meet-and-greets online
   - Read success stories
   - Access post-adoption resources

8. **Open Response:** What would make pet adoption easier in Ashtabula County?
   [Text field]

**Survey Tool:** Google Forms (free) or Typeform (free tier)
**Estimated Completion:** 3 minutes
**Incentive:** None (community goodwill) or enter to win $25 pet store gift card

---

### User Interviews (Target: 5-8 participants)

**Recruitment:**
- Survey respondents who opt-in
- ACAPL adoption event attendees
- Facebook group volunteers

**Interview Script (30 minutes):**

1. **Background:** Tell me about your pet adoption experience (if any)
2. **Current Process:** Walk me through how you'd search for a pet today
3. **Pain Points:** What frustrates you about current options?
4. **Personality Matching:** Would you trust an algorithm to suggest compatible pets? Why/why not?
5. **Feature Priorities:** Which features would you use first?
6. **Barriers:** What would stop you from using this tool?

**Compensation:** $20 gift card (optional, budget-dependent)

### Petfinder API Details

**Endpoint:** `https://api.petfinder.com/v2/animals`
**Authentication:** OAuth 2.0 (client credentials flow)
**Rate Limits:** 1,000 requests/day (free tier)
**Ashtabula Filter:** `location=44004&distance=25`

**Sample Response Structure:**
```json
{
  "animals": [
    {
      "id": 12345,
      "name": "Buddy",
      "type": "Dog",
      "breeds": { "primary": "Labrador Mix" },
      "age": "Adult",
      "gender": "Male",
      "size": "Large",
      "photos": [...],
      "contact": { "email": "adopt@acapl.org" },
      "organization_id": "OH123"
    }
  ]
}
```

**Note:** Petfinder API does NOT include personality data — we'll need shelter-side assessment for matching.

### Personality Assessment Framework

We'll use the **ASPCA Meet Your Match** methodology (publicly documented):

**Canine-ality/Feline-ality Dimensions:**
1. **Sociability** (1-5 scale)
2. **Playfulness** (1-5 scale)
3. **Chill Factor** (1-5 scale)
4. **Trainability** (1-5 scale)
5. **Independence** (1-5 scale)

**Shelter Assessment Time:** ~2 minutes per pet  
**Training Required:** 1-hour volunteer/staff orientation
