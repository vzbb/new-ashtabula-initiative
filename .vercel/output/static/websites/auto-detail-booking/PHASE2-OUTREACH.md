## 4. Detailer Onboarding Process

### Setup Checklist

**Week 1: Profile Creation**
- [ ] Business information (name, address, service area)
- [ ] Upload logo/photos
- [ ] Create service packages (3-5 tiers)
- [ ] Set pricing by vehicle type
- [ ] Define availability (hours, days)
- [ ] Add team members (if applicable)

**Week 2: Customization**
- [ ] Write service descriptions
- [ ] Upload package photos
- [ ] Configure add-ons
- [ ] Set deposit amount ($20-50 typical)
- [ ] Add booking policies
- [ ] Connect payment account

**Week 3: Testing**
- [ ] Test booking flow
- [ ] Test payment processing
- [ ] Test SMS notifications
- [ ] Train staff (if applicable)
- [ ] Soft launch with existing customers

**Week 4: Launch**
- [ ] Add booking link to Facebook/Instagram
- [ ] Update Google Business Profile
- [ ] Email existing customers
- [ ] Monitor first bookings
- [ ] Gather feedback

## 5. Risk Assessment Matrix

| Risk | Likelihood | Impact | Mitigation |
|------|------------|--------|------------|
| Detailers don't see value | Medium | HIGH | Free tier, testimonials from early adopters |
| Preference for cash payments | Medium | MEDIUM | Emphasize deposit reduction of no-shows |
| Technological resistance | Medium | MEDIUM | Simple setup, phone support |
| No-show reduction not realized | Low | HIGH | Deposit requirement, reminders |
| Competing platform launches | Low | MEDIUM | First-mover, local focus |

## 6. Go/No-Go Decision Tree

```
After outreach:
│
├─► If 3+ detailers commit to pilot:
│   └─► Proceed to Phase 3 (SPEC.md)
│       └─► Build with detailer input
│
├─► If 1-2 detailers interested:
│   └─► Proceed with limited pilot
│       └─► Focus on their specific needs
│
├─► If Chamber partners:
│   └─► Leverage for recruitment
│       └─► Member benefit positioning
│
└─► If minimal interest:
    └─► Pause
        └─► Reassess pricing/feature set
```

### Go Criteria
- [ ] 3+ detailers commit to using platform
- [ ] OR 1 shop with 2+ employees commits
- [ ] OR Chamber agrees to partnership

### No-Go Criteria
- [ ] Detailers satisfied with phone/Facebook
- [ ] No interest in online payments
- [ ] Major competitor enters market
