# Rental Availability Finder

AI-powered rental matching tool for Ashtabula County, Ohio. Connect renters with available properties faster through intelligent budget-based recommendations.

## Features

- 🔍 **Budget-Based Matching** - Enter max monthly rent for personalized results
- 🤖 **Gemini AI Summaries** - Smart rental overviews with next-step guidance
- ⚡ **Instant Results** - Get rental options in seconds, not hours
- 🏠 **Local Focus** - Specialized for Ashtabula County market

## Use Cases

- **Renters** - Find affordable housing within your budget
- **Landlords** - Reduce vacancy time with better renter matching
- **Property Managers** - Streamline tenant placement process
- **Social Services** - Assist clients with housing searches

## Tech Stack

- React 18 + Vite
- Google Gemini 1.5 Flash API
- CSS3 with custom properties

## Environment Variables

```bash
VITE_GEMINI_API_KEY=your_gemini_api_key
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Demo Script

1. Enter maximum monthly budget (e.g., $850)
2. Click "Find Rentals"
3. AI generates 3 bullet summary of available options in budget range
4. Use summary to guide rental search or share with clients
