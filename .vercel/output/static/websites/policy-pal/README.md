# Policy Pal

AI-powered insurance policy summarizer for insurance agents. Transform complex policy documents into clear, client-friendly summaries that improve retention and renewal conversations.

## Features

- 🤖 **Gemini AI Summarization** - Converts dense policy language into 3-bullet highlights
- ⏱️ **5-Second Output** - Instant summaries for client calls and emails
- 🛡️ **Review CTA Generator** - Automatically includes coverage review prompts
- 📋 **Policy-Aware** - Handles renewals, endorsements, and coverage changes

## Use Cases

- **Renewal Conversations** - Give clients clear reasons to stay
- **New Business** - Explain coverage options in plain English
- **Claims Prep** - Pre-claim coverage summaries reduce disputes
- **Email Follow-ups** - Professional summaries for client records

## Tech Stack

- React 18 + Vite
- Google Gemini 1.5 Flash API
- CSS3 with custom properties

## Environment Variables

```bash
VITE_GEMINI_API_KEY=your_gemini_api_key
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Demo Script

1. Enter policy details (e.g., "Homeowners policy renewal with 5% rate increase and new water backup coverage")
2. Click "Generate Summary"
3. AI produces 3 bullet highlights + coverage review CTA
4. Copy and send to client or use in renewal call
